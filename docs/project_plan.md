## Overview
This document tracks the development of the ESP32-based smart lock, alarm, and proximity system.

## Goals
- Automatic unlock based on phone proximity
- Auto-lock when phone leaves range or engine is running
- Optional soft immobilizer
- Alarm system integration
- Maintain full factory fallback (keys always work)

2. Wiring diagram
3. BLE implementation
4. Relay control
5. Sensor integration
6. Power management & deep sleep
7. Security model
8. Testing & validation
9. Installation in vehicle
